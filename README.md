# dynamic-script
演示：使用Grunt-contrib-connect为html动态添加script标签

clone/下载本项目后，请在根目录下执行
```
npm install 
```
然后执行
```
grunt
```
就会调用浏览器打开一个地址，如果你能看到一个alert,内容是'I am from dynamic script',就说明动态插入script成功了！
